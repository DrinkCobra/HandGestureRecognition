This program is developed for Hand Gesture Recognition through the utilization of MediaPipe and OpenCV. 

The program can detect the following gestures:
	*okay
 	*peace
	*thumbs up
	*thumbs down
	*call me
	*stop
	*rock
	*live long
	*fist
	*smile
